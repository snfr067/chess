# 台灣暗棋 PWA mobile-r8

本版重點：

- 玩家與 AI 共用同一套可視動作流程。
- 食用暗棋會先翻開、停留展示，再判定食用是否成功。
- 成功食用後會顯示目標格、被翻出的棋、進入墳墓的棋。
- 墳墓新增棋會高亮跳出。
- 非法操作只用小通知提示，不再用大量文字訊息取代 UI。
- 設定頁含 AI 難度、連吃規則、AI 每步延遲。
- AI 不讀取暗棋真實內容，僅以公開資訊與剩餘棋種機率估算。

## 本機測試

在此資料夾執行：

```powershell
py -m http.server 8000
```

開啟：

```text
http://localhost:8000/
```

## GitHub Pages

把本資料夾內檔案全部放到 repository 根目錄，Pages 設定為 main / root。


## v9 橫向限定

這版已改成手機橫向限定：直向手機畫面會顯示「請將手機打橫」，不再嘗試直向堆疊棋盤與資訊區。manifest 也設定為 landscape，加入主畫面後會以橫向為主要使用方式。
